< section viewer > controls
slider

  • The slider controls comprise, from left to right
    • a label, indicating the parameter to be changed
    • a slider bar
    • a text box which displays the current value of the parameter
  • The slider may be used to adjust the appropriate parameter value in one of the following ways:
    • click on the slider knob and drag it to the left or right
    • click in the slider channel (not on the knob). This will adjust the value of the parameter by +/- 1 (depending on whether the slider is clicked to the right or left of the knob).
    • if the mouse button is held down (in the channel next to the knob) the value of the parameter will adjust continuously by an increment of +/- 1.
    • edit the value in the text box. Note that the return or enter key must be pressed for the edit to take effect.
  • The section viewer has the following slider controls available:
    • dist
      • adjusts the distance of the section from a fixed point.
    • yaw
      • adjusts rotation around the vertical axis.
    • pitch
      • adjusts rotation around the horizontal axis, (side to side).
    • roll
      • adjusts rotation around the horizontal axis, (in and out of screen).
    • user defined rotation (not available yet)
      • adjusts rotation around a user defined axis.
