point entry

  • A point entry dialogue will be opened if the user elects to change the fixed point by typing coordinates (instead of using the mouse).
  • This is achieved by selecting the following menu entry ...
    fixed point/change fixed point/typing coordinates
  • The point entry dialogue displays the current values of X, Y and Z for the fixed point.
  • The X, Y and Z values relate to the 3D image (not the 2D section).
  • If a user wishes to change the position of the fixed point the appropriate fields in the point entry dialogue must be edited and then the OK button clicked.
