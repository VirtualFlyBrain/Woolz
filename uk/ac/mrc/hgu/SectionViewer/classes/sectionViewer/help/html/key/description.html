Anatomy Key

The Anatomy key indicates the colour of anatomy components in the section viewer. It also gives the (fully qualified) name of the component.

The Anatomy Key will launched when an item of anatomy is selected from a menu in the main application (that is using the section viewer).

Only one Key will be launched.

The Key has 6 rows, each comprising ...

  • a coloured panel
  • left/right arrow buttons
  • a scrollable text field
  • a visibility checkbox
  • a zap/replace button

The 6 preset colours are ...

  1. brown
  2. red
  3. yellow
  4. blue
  5. green
  6. cyan

If more than 6 items of anatomy are selected, the original items are replaced starting with item #1.
