colour button

  • The colour button is located at the top left of the section view window (to the left of the text box which displays the name of the 3D Woolz file).
  • The colour of the button matches the colour of the section representation in the 3D viewer.
  • Clicking on the button will open up a colour chooser dialogue which allows the user to change the colour of the button if required. This will also change the colour of the section representation in the 3D viewer.
