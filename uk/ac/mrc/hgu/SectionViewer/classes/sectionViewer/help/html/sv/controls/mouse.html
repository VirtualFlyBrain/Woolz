mouse

The effect of clicking and/or dragging the mouse in the grey-level image varies depending upon the options that have been selected in the config menu.
The panel which displays feedback information may be made visible/invisible by toggling the menu item config/show cursor feedback.
Position and grey-level feedback information for the current cursor position are always available (if the feedback panel is visible).
Anatomy feedback for the current cursor position is only available if the menu item config/enable 'mouse-click' anatomy is selected (and the feedback panel is visible).
