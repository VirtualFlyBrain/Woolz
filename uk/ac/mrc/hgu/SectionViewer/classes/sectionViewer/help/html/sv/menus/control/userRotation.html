control/rotation/user-defined

a checkbox which toggles the visibility of a (rotation) slider control ...

  • displays a panel at the bottom of the section viewer which allows the user to perform a rotation of the 'view structure' about an arbitrary axis of rotation.
  • one end of this arbitrary axis of rotation is set using the 'fixed point' menu items.
  • the other end is set using the 'rotation axis' menu items.
