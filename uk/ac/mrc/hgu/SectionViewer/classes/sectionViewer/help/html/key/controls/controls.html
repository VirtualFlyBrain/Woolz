< anatomy key > controls

Each row in the anatomy key has its own set of controls.
From left to right, starting next to the coloured patch the controls are ...

  • left/right arrow buttons
    • Clicking on one of these arrow buttons scrolls the text field fully to the left or right respectively.
  • scrollable text field
    • This displays the (fully qualified) name of the anatomy component.
    To scroll left or right, press the left mouse button in the text field and drag the mouse left or right respectively.
  • visibility checkbox
    • This checkbox toggles the visibility (highlighting) of an anatomy component (in section views).
  • zap/replace button
    • Clicking on this button when the zap (lightning) symbol is displayed will remove the anatomy component from the key. (The replace (arrow) symbol will then appear.) The component will no longer be highlighted in section views.

    Clicking on this button when the replace symbol is displayed will replace the anatomy component in the key. (The zap symbol will re-appear.) The component will be highlighted again.

    When a replace symbol is displayed, that slot in the key will be used if a further anatomy selection is made from the menu. The original anatomy component will no longer be available (unless it is re-selected from the menu).
