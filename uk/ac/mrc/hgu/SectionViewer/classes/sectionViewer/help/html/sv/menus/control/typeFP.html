control/fixed point/change by typing coordinates

If this menu item is selected:

  • A point entry dialogue will be openend.
  • A new fixed point may be defined by editing the X Y Z fields of the point entry dialogue.
  • The X Y Z coordinates relate to the underlying 3D image, not the 2D image.
  • The new fixed point will be displayed as a pink cross within a circle.
  • The distance slider will be reset so that 0 corresponds to the new fixed point
