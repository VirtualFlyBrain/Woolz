view mode

The view mode describes how a 2D section through the 3D image is projected onto a viewing surface (such as the computer screen).
In practice, the two view modes available in this programme may be differentiated by their effect on the roll angle ...

  • up is up mode calculates the roll angle automatically. When this mode is selected it is not possible to manually change roll angle.
  • absolute mode allows the user to manually change roll angle.
