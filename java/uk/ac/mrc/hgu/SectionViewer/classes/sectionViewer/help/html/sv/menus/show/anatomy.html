show/'mouse-click' anatomy

This menu item has a checkbox which toggles the visibility of anatomy feedback obtained by clicking or dragging the mouse in the 2D image.

  • anatomy feedback is only available if anatomy components have been painted in the underlying 3D image.
  • if an anatomy component exists at the mouse position it will be highlighted in magenta.
  • the fully qualified name of the anatomy component will be displayed in the anatomy feedback area.
  • the text in the anatomy feedback area may be scrolled by clicking and dragging the mouse on the text.
