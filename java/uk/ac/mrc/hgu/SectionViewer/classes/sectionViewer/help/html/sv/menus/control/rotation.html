< section viewer > menus
control/rotation/yaw, pitch, roll

This menu item has a checkbox which toggles the visibility of the yaw pitch roll controls

  • It displays a set of sliders at the bottom of the section viewer which allows the user to adjust the following angles in the view structure
    • yaw (rotation around the z axis)
    • pitch (rotation around the x axis)
    • roll (rotation around the y axis)
  • roll is only enabled if view mode is absolute.
