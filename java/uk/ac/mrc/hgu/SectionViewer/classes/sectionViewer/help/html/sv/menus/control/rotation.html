control/rotation/yaw, pitch, roll

a checkbox which toggles the visibility of (rotation) slider controls ...

  • displays a panel at the bottom of the section viewer which allows the user to adjust the following angles in the 'view structure' ...
    • yaw (rotation around the z axis)
    • pitch (rotation around the x axis)
    • roll (rotation around the y axis)
  • roll is only enabled if the 'view mode' is 'absolute'.
