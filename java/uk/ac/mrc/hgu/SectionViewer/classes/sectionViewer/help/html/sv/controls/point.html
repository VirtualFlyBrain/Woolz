point entry

  • A point entry dialogue will be opened if the user elects to change the fixed point or rotation axis by typing coordinates (instead of using the mouse).
  • The point entry dialogue displays the current values of X, Y and Z for the fixed point.
  • The X, Y and Z values relate to the underlying 3D image (not the 2D section).
  • The appropriate fields in the point entry dialogue must be edited and the OK button clicked before any change takes effect.
