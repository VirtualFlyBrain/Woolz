control/fixed point/change using mouse button

If this menu item is selected:

  • the mouse cursor will change from an arrow shape to a cross and wherever the mouse is next clicked (inside the image window) will define the new fixed point.
  • The new fixed point will be displayed in this section as a pink cross within a circle.
  • The distance slider will be reset so that 0 corresponds to the new fixed point
  • the cursor will revert to an arrow shape.
