< section viewer > menus
control

The control menu allows users to manipulate the view structure of the underlying 3D woolz object and so change the displayed section.

The following menus are available:

  • rotation
    • yaw, pitch roll
    • user-defined
  • view mode
    • up is up
    • absolute
  • fixed point
    • change using mouse
    • change by entering coordinates
    • reset
  • rotation axis
    • change using mouse
    • change by entering coordinates
    • reset
  • reset controls
