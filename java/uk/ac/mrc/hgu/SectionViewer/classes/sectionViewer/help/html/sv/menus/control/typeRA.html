control/rotation axis/change by typing coordinates

If this menu item is selected:

  • A point entry dialogue will be openend.
  • A new rotation axis may be defined by editing the X Y Z fields of the point entry dialogue.
  • The X Y Z coordinates relate to the underlying 3D image, not the 2D image.
  • The fixed point is the other point which defines the rotation axis.
  • This menu item is currently not available.
