Section Viewer

The Section Viewer is a software component ( java bean) that can be dropped into a java application in order to display 2D sections through 3D Woolz images.

Applications will typically instantiate the SVParent class which manages the creation of section viewers. The SVParent class has methods which will create section viewers in either external or internal windows (JFrame or JInternalFrame) respectively.

Applications which are capable of displaying a 3D image will typically extend SVParent so that changes to the section may be reflected in the 3D display. An example of such an extension is SVParent3D which is used with the JWlzViewer application.

Controls in the Section Viewer allow the user to manipulate the position of the section relative to the underlying 3D image. Any arbitrary section is obtainable. The section viewer uses native Woolz libraries (written in C) to calculate each new section.

The cursor (mouse) position in the 2D image provides feedback of the x,y,z coordinates in the underlying 3D image. It also gives the grey level value at this position.

If the underlying 3D image has anatomy components associated with it, it is possible to highlight these in the grey-level image. This can be achieved in two ways:

  • by selecting anatomy components from a menu in the main application.
  • by clicking the mouse in the 2D image.
